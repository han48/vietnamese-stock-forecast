🔬 ARIMAX — ARIMA with eXogenous Features

Sử dụng features tính từ OHLCV: Log Return, H-L Range, Volume/MA5, RSI 14, MACD Histogram, VAR(1) prediction (open, high, low, volume → close proxy). Cùng mã & khoảng thời gian đã chọn ở trên.

🔍 Giải thích features OHLCV + VAR
FeatureCông thứcÝ nghĩa
returnsln(close_t / close_{t-1})Log return — xu hướng giá
hlRange(high − low) / closeBiên độ giá trong phiên
volumeMa5volume / MA5(volume)Áp lực khối lượng so với trung bình
rsi14RSI(14) / 100Động lượng giá (0–1)
macdHist(MACD − Signal) / closeTín hiệu xu hướng ngắn hạn
varPredVAR(1)[open,high,low,vol] → proxy / close − 1Dự báo 1 bước từ Vector Autoregression

ARIMAX = OLS(close ~ features) + ARIMA(p,d,q) trên phần dư (residuals).
